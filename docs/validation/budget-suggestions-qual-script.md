# Roteiro qualitativo — Sugestões de orçamento (v1)

**Participantes:** 3–5 pessoas com dados reais no Saldo Real  
**Duração:** ~15 min por sessão  
**Facilitador:** observa em silêncio; anota tempo e falas espontâneas

## Preparação

| Cenário | Como montar |
|---------|-------------|
| A — Sem dados | Perfil novo ou backup vazio; zero saídas |
| B — ~7 dias | Import parcial ou lançamentos manuais só na última semana |
| C — 90+ dias | CSV completo com ≥8 categorias de saída distintas |

## Roteiro

1. **Contexto (1 min)**  
   "Queremos facilitar criar o primeiro limite mensal por categoria. Tudo fica no seu aparelho."

2. **Tarefa principal (cenário C)**  
   - Abrir `/config?tab=orcamentos` ou `/dashboard?tab=orcamentos`  
   - Usar **Sugerir com base no meu histórico**  
   - Criar pelo menos 1 orçamento sem ajuda  
   - **Medir:** tempo do clique no CTA até confirmação

3. **Probes após criar**  
   - "Os valores fazem sentido para você?"  
   - "Você mudou algum valor antes de salvar? Por quê?"  
   - "O texto sobre mediana/meses ficou claro?"

4. **Cenário B (fallback)**  
   - Repetir abertura do drawer  
   - Esperado: mensagem de dados insuficientes + atalho manual  
   - **Anotar:** frustração ou compreensão da mensagem

5. **Cenário A (opcional)**  
   - Confirmar que só aparece CTA manual + nota sobre 30 dias

6. **Follow-up (opcional, +30 dias)**  
   - Orçamentos criados ainda estão ativos?  
   - Algum foi desativado? Motivo?

## Critérios de sucesso

- ≥3 de 5 participantes (cenário C) criam ≥1 orçamento via sugestão sem ajuda  
- Nenhum descreve a sugestão como "errada" ou "inventada"  
- Tempo mediano até primeiro orçamento &lt; 60 s (cenário C)
